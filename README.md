# java-carRace
KTC step1 과제1 자동차경주 구현

## 프로젝트 소개
- 자동차 이름 및 갯수 시행 횟수 입력 받음.
- 0-9의 무작위 수중 4이상일 경우 한 칸 전진.
- 최종우승자 출력(여러명 가능)

## Car 객체
- **name** : 자동차 이름
- **go** : 전진 횟수
- **compareTo** : go 비교
  
## 기능 소개
- **CarRaceTry** : 각 차마다 랜덤숫자가 4이상이면 go를 1 진전시킨다.
- **CarRacePrint** : 시행 횟수마다 결과 출력.
- **printGo** : go만큼 '-' 출력.
- **jointWinner** : 공동 우승이 있을시 이름 출력.
